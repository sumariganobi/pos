## ğŸ“ Design System

### Typography
```
- Headings: text-xl (20px)
- Body: text-sm (14px)
- Small: text-xs (12px)
- Font weight: font-medium, font-semibold, font-bold
```

### Spacing
```
- Padding: p-3, p-4, p-6
- Margin: mb-2, mb-3, mb-4, mb-6
- Gap: gap-2, gap-3, gap-4
```

### Colors
```
- Primary: Blue (#2563eb) to Purple (#9333ea)
- Success: Green (#10b981)
- Warning: Yellow/Orange (#f59e0b)
- Danger: Red (#ef4444)
- Gray: 50-900 scale
```

### Components
```
- Cards: rounded-lg, shadow-sm, hover:shadow-md
- Buttons: rounded-lg, py-2, px-3, text-sm
- Inputs: rounded-lg, py-2, px-3, text-sm
- Badges: rounded-full, px-2, py-0.5, text-xs
```

### Responsive Breakpoints
```
- Mobile: < 640px (sm)
- Tablet: 640px - 1024px (md, lg)
- Desktop: > 1024px (lg, xl)
```
